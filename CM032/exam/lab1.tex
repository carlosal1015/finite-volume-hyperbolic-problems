	\question

	Cuando hablamos de punto flotante, discutimos el
	\textit{épsilon de la máquina}, $\epsilon$ -este es el número más
	pequeño que cuando se suma a $1$ sigue siendo diferente de $1$.
	Calcularemos $\epsilon$ aquí:
	\begin{itemize}
		\item Pick an initial guess for $\epsilon$ of \mintinline{python}|eps = 1|.
		\item Create a loop that checks whether \mintinline{python}|1 + eps| is different from \mintinline{python}|1|.
		\item Each loop iteration, cut the value of \mintinline{python}|eps| in half.
	\end{itemize}
	What value of $\epsilon$ do you find?

	\question

	To iterate over the tuples, where the \textit{i}th tuple contains
	the \textit{i}-th elements of certain sequences, we can use
	\mintinline{python}|zip(*sequences)| function.

	We will iterate over two lists, \mintinline{python}|names| and
	\mintinline{python}|age|, and print out the resulting tuples.

	\begin{itemize}
		\item

		      Start by initializing lists
		      \mintinline{python}|names = ["Mary", "John", "Sarah"]| and
		      \mintinline{python}|age = [21, 56, 98]|.

		\item

		      Iterate over the tuples containing a name and an age, the
		      \mintinline{python}|zip(list1, list2)| function might be
		      useful here.

		\item

		      Print out formatted strings of the type
		      \mintinline{text}|"*NAME is AGE years old*"|.
	\end{itemize}

	tuples containing indices of objects in the sequence, and the
	objects.

	with the random numbers.

	generates a random number not smaller than
	\mintinline{python}|start|, and not bigger than
	\mintinline{python}|end|.

	\begin{itemize}
		\item

		      Generate a list of $10$ random numbers from $0$ to $9$.

		\item

		      Using the \mintinline{python}|enumerate(random_list)|
		      function, iterate over the tuples of random numbers and
		      their indices, and print out
		      \mintinline{text}|"Match: NUMBER and INDEX"| if the random
		      number and its index in the list match.
	\end{itemize}
